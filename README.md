# Embedded Systems & Automation Design Portfolio

A personal showcase of embedded systems programming concepts, demonstrated through interactive Wokwi simulations. Built as part of a university assignment for Group 4220z at Riga Nordic University.

## 🚀 Live Demo

https://alanrzii.github.io/robotic-task/embedded-systems-portfolio.html *(Note: This link will work after GitHub Pages finishes deploying)*

## 🛠️ Tasks & Simulations Included

This project explores fundamental embedded software and hardware concepts through 6 specific scenarios using the Arduino UNO microcontroller:

1. **Basic Output Control** 💡
   - Simple digital output to blink an LED.
2. **Variable Control System** 🎛️
   - PWM brightness control using an analog potentiometer.
3. **User Interaction System** 🔘
   - Push-button input to toggle an LED with software debouncing.
4. **Alert & Notification System** 🔔
   - Audible triggered alerts using a piezo buzzer.
5. **Display & Information System** 📺
   - Displaying statuses and counters on a 16x2 I2C LCD.
6. **Environmental Sensing System** ☀️
   - Automated LED lighting response using a Light Dependent Resistor (LDR).

## 💻 Technologies Used
- **HTML, CSS, JS:** Custom-built frontend with a Synthwave/Cyberpunk aesthetic.
- **Wokwi:** Embedded simulation platform for running Arduino C++ code directly in the browser.
- **Arduino C++:** Core logic written for AVR microcontrollers.
